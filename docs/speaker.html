Reproducible Research with Table

If you are a data scientist or researcher, you will certainly be interested in reproducible research. R package 'rrtable' makes it possible to make reports with HTML, LaTex, MS word or MS Powerpoint formats from a table of R codes. In the package, modularized 'shiny' app codes are provided. These modules are intended for reuse across applications.

이남신 오피지지

실제 기업 서비스에서 R 활용하기

R은 데이터를 가공, 분석하기에 굉장히 편리한 언어입니다. 하지만 기업에서 데이터 분석가는 분석 모델을 설계하는 것에 그치지 않고, 이를 실제로 운영되는 서비스에 적용시키는 것까지 고려해야 합니다. 또한 다른 동료와 협업하는 경우 서로의 작업 결과를 재현할 수 있도록 개발 환경을 동기화하는 것 역시 중요합니다. 이번 발표에서는 Docker와 Kubernetes를 활용하여 R로 쓰인 분석 모델을 기업의 라이브 서비스에 적용하고, 동료와 공유하는 방법을 말씀드리려 합니다.

박성우 베가스

데이터 옵스의 시작 데이터사이언스캔버스와 그 활용

데이터 경제, 산업의 시대 기업 의사결정의 핵심요소가 데이터 & 데이터분석이 중심이 되고 있으며, 데이터옵스 협업도구로 데이터사이언스캔버스와 활용을 소개 하고자 합니다. 데이터사이언스캔버스는 10여년동안 제조 ·공공 ·금융 등 산업현장에서 데이터분석 프로젝트, 기업 재직자양성교육, 데이터 경제·산업시장에 대한 연구 경험과 노하우를 해당 데이터 분석 관련 업무 전문가, 엔지니어, (시티즌)데이터과학자가 쉽고 빠른 교육과 협업을 통한 업무를 진행 할 수 있도록 데이터 분석 방법론 및 가이던스 체계를 개발한 데이터과학 도구 입니다.

이영록 The Walt Disney Company

데이터분석가의 R 패키지개발도전기

R로데이터분석을하고계신분이라면, R 패키지를개발할이유가생긴것이고또한준비가된것입니다. R 패키지를만드는일은개발자의영역으로만생각했던데이터분석가가처음으로 R 패키지제작작업을시작하게된동기와, 그작업과정에서배운점들을소개하고자합니다.
